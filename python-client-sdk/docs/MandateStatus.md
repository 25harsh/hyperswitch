# MandateStatus

The status of the mandate, which indicates whether it can be used to initiate a payment.

## Enum

* `ACTIVE` (value: `'active'`)

* `INACTIVE` (value: `'inactive'`)

* `PENDING` (value: `'pending'`)

* `REVOKED` (value: `'revoked'`)
